- **50% boost** in semantic search relevance via intelligent chunking strategies
- **40% reduction** in manual Jira ticket triage
- **2,300+ Jira items** successfully indexed and queried

### 🔷 Diabetic Retinopathy Detection — CNN + ResNet50

- **93.6% test accuracy** on holdout set

- **96.65% combined accuracy**
- Published in AIM 2025 (Scopus-indexed)
